# Agro Forte, Futuro Sustentável: Equilíbrio entre Produção e Meio Ambiente

## 🌾 Descrição do Projeto
Este site foi desenvolvido como projeto de avaliação para o **Concurso Agrinho 2026**, na Categoria Programação (Subcategoria 3 - Ensino Médio). O objetivo da aplicação é conscientizar sobre a importância do equilíbrio entre a produtividade agrícola e a conservação ambiental, apresentando práticas modernas e um simulador interativo de tomada de decisões.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica de conteúdo.
* **CSS3:** Estilização moderna, uso de variáveis globais, Grid Layout e Media Queries para responsividade total.
* **JavaScript (Vanilla):** Manipulação dinâmica do DOM através de escutadores de eventos assíncronos (`addEventListener`), sem códigos inline.
